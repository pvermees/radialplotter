Colours

The Options → Colours menu item allows the user to customise the colours of the graphical output. The Colours dialog offers different options depending on whether the Density Plot or Radial Plot option has been selected in the Output menu.

Density Plots

The edge and fill colour of each of the four graphical elements (KDE, PDP, Histogram and Points) selected in the Options → Settings dialog can be specified. These colours are transparent so that even if all elements are filled, the underlying objects remain visible.

Radial Plots

The radial plot input window has one extra column, in which the user can specify a colour index. This can be used, for example, to show kinetic indicators such as Dpar or Cl content of fission track data. If the third data column is not empty, the graphical output will include a colour bar. This colour bar consists of 100 steps, calculated by linear interpolation between the minimum and maximum values of the third data column. The minimum and maximum colour can be set by opening the Options → Colours menu.
