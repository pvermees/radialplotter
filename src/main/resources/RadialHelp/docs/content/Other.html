In addition to fission track counts using the external detector method, any kind of other heteroscedastic data can be plotted as well, such as U-Pb ages, LA-ICP-MS based fission track ages or luminescence doses. The input file of these 'other' data consists of a line with the sample name and an optional flag ('O' for Other) followed by n lines of data (x), analytical uncertainties (σ(x)), and an optional colour code.
